第一章学习：
  2**3 ： 表示2的3次方
  n = 10; `n`:反引号表示把值转为字符串，同str、repr相同，字符串不能直接同数字拼接
  import math : 引入模块方式
  from math import floor :表示引入math 模块的 floor 方法，使用的时候不需要加上 math. 不推荐
  1/2 :结果为 0，默认直接取整，如需要显示浮点数 在除数和被除数改成浮点数： 1/2.0 = 0.5
  1000L 长整数后面加L
  '''hello''' : '''表示长字符串，可跨多行书写
  // 整除 python 2.2 之后版本可用
  新的函数：
    1. abs(number) 返回数字绝对值
    2. cmath.sqrt(number) 返回平方根可用于负数
    3. math.sqrt(number) 返回平方根 不可用于负数
    4. int() 将字符串和数字转换为整数
    5. float() 将数字和字符串转换为浮点数
    6. long() 将数字和字符串转换为长整型数
    7. pow(x,y) 返回x的y次幂
    8. str() 将值转换为字符串
    9. repr() 返回值的字符串表示形式
    10.round(number,n) 根据给定的精度对数字进行四舍五入
    11.math.ceil(number) 返回数的向上取整，返回浮点数类型
    12.math.floor(number) 返回数的向下取整，返回浮点数类型
    13.raw_input() 获取用户输入，结果被看着原始字符串
    14.input() 获取用户输入，用户输入需要自己加引号，不推荐使用
